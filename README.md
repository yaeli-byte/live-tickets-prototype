# Live Tickets — Prototype

RTL Hebrew mobile ticketing prototype (Home · Category · Event Lobby · Event · Search + occasion collection), built with the homepage design system.

**Live prototype:** open `flow.html` (the root `/` redirects there).

- `flow.html` / `flow.css` — the connected prototype (single-file vanilla-JS SPA)
- `assets/` — icons, logo, and imagery
- `homepage.html` — the original standalone homepage
- `design-system/` — extracted tokens

Best viewed at a 375px mobile width.
